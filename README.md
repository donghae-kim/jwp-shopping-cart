- [x] 상품

    - [x] 이미지
    - [x] 가격
        - [x] 숫자로만 이루어진다.
        - [x] 0원 이상 1억원 이하이다.
        - [x] 10원 단위여야 한다.

- [x] Product CRUD

### Service

- [x] 장바구니 서비스
    - [x] 상품 추가

### API 명세

- [x] 상품 Create
    - POST "/product"
    - Request: name, price, image
    - Response: 200 OK
- [x] 상품 Read
    - GET "/products"
    - Response: products{ {id, name, image, price}, {..}}
- [x] 상품 Update
    - PUT "/product"
    - Request: id, name, price, image
    - Response: 200 OK
- [x] 상품 Delete
    - DELETE "/product/{id}"
    - Response: 200 OK
